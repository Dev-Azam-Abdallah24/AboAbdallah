1 Lecture 1: Introduction to Artificial Intelligence

تاريخ الذكاء الاصطناعي

1956 - ولادة المصطلح
صياغة مصطلح "الذكاء الاصطناعي" في مؤتمر دارتموث، بداية البحث الأكاديمي المنظم في هذا المجال.
1980-1974 - الشتاء الأول
فترة تراجع في التمويل والاهتمام بسبب التوقعات المبالغ فيها وعدم تحقيق النتائج المرجوة.
1989 - نظرية التقريب الشامل
إثبات قدرة الشبكات العصبية على التقريب لأي دالة مستمرة مما وضع أساساً لنظرية التعلم.
1987-1993 - الشتاء الثاني
فترة أخرى من التراجع بعد إفلاس سوق أجهزة الكمبيوتر المتخصصة في الذكاء الاصطناعي.
2010s - ثورة التعلم العميق
بداية الثورة المدعومة بوجود معالجات الرسوميات (GPUs)، مما أتاح طفرة نوعية في مجالات الصور والكلام.
2020s - الذكاء الاصطناعي التوليدي
ظهور نماذج Transformer ونماذج اللغة الضخمة مثل DALL-E وChatGPT مما أحدث طفرة في فهم وإنتاج المحتوى.
Artificial Intelligence (الذكاء الاصطناعي)
Computer systems that can simulate tasks that typically require human intelligence, like learning, problem-solving, and decision-making.
أنظمة حاسوبية تحاكي المهام التي تتطلب عادة ذكاءً بشرياً مثل التعلم وحل المشكلات واتخاذ القرار.
[AI_introCh1.pdf, p. 2]
Agent / Intelligent Agent (العامل / العامل الذكي)
A machine that solves problems that are challenging for humans. Anything that can be viewed as perceiving its environment through sensors and acting upon that environment through actuators.
آلة تحل مشاكل يصعب على البشر حلها. أي شيء يمكن اعتباره يدرك بيئته عبر أجهزة الاستشعار ويتصرف فيها عبر المحركات.
[AI_introCh1.pdf, p. 4]
Narrow AI (الذكاء الاصطناعي الضيق)
Focuses on intelligent agents to solve a specific problem.
يركز على عوامل ذكية لحل مشكلة محددة.
[AI_introCh1.pdf, p. 4]
Artificial General Intelligence (AGI) (الذكاء الاصطناعي العام)
A hypothetical intelligent agent which can understand or learn any intellectual task that human beings can.
عامل ذكي افتراضي يمكنه فهم أو تعلم أي مهمة فكرية يستطيع الإنسان القيام بها.
[AI_introCh1.pdf, p. 4]
Turing Test (اختبار تورنج)
A test that tries to define what acting like a human means. It assesses a machine's ability to exhibit intelligent behavior equivalent to, or indistinguishable from, that of a human.
اختبار يحدد معنى التصرف كإنسان. يقيس قدرة الآلة على إظهار سلوك ذكي مماثل أو لا يمكن تمييزه عن الإنسان.
[AI_introCh1.pdf, p. 6]
Natural Language Processing (NLP) (معالجة اللغات الطبيعية)
An AI area focused on enabling computers to understand, interpret, and generate human language.
مجال يهتم بتمكين الحاسوب من فهم وتفسير وتوليد اللغة البشرية.
[AI_introCh1.pdf, p. 6]
Knowledge Representation (تمثيل المعرفة)
An AI area concerned with how knowledge is stored and organized in a way that allows AI systems to use it for reasoning.
مجال يهتم بكيفية تخزين وتنظيم المعرفة بحيث يمكن للأنظمة استخدامها للاستدلال.
[AI_introCh1.pdf, p. 6]
Automated Reasoning (الاستدلال الآلي)
An AI area focused on enabling computers to draw conclusions from existing knowledge using logical inference.
مجال يركز على تمكين الحاسوب من استخلاص النتائج من المعرفة الموجودة باستخدام الاستدلال المنطقي.
Actuators (المؤثرات)
The mechanisms through which an agent acts upon its environment.
الآليات التي يستخدمها العامل للتأثير على البيئة.
Sensors (أجهزة الاستشعار)
The mechanisms through which an agent perceives its environment.
الآليات التي يستخدمها العامل لإدراك البيئة.
Percept (الإدراك)
The agent's perceptual inputs at any given instant.
مدخلات العامل الحسية في أي لحظة معينة.
Percept Sequence (تسلسل الإدراك)
The complete history of everything the agent has ever perceived.
التاريخ الكامل لكل ما أدركه العامل على الإطلاق.
Agent Function (دالة العامل)
A mathematical function that maps from the set of all possible percept sequences P* to the set of actions A (f: P* → A).
دالة رياضية تربط جميع تسلسلات الإدراك الممكنة بالإجراءات (f: P* → A).
[AI_agentsCh2.pdf, p. 4]
Agent Program (برنامج العامل)
A concrete implementation of the agent function for a given physical system.
تنفيذ ملموس لدالة العامل لنظام مادي معين.
[AI_agentsCh2.pdf, p. 4]
Rational Agent (العامل العقلاني)
For each possible percept sequence, a rational agent should select an action that maximizes its expected performance measure, given the evidence provided by the percept sequence and the agent's built-in knowledge.
لكل تسلسل إدراك ممكن، يجب أن يختار العامل العقلاني إجراءً يزيد من مقياس الأداء المتوقع بناءً على الأدلة والمعرفة المدمجة.
[AI_agentsCh2.pdf, p. 9]
Performance Measure (مقياس الأداء)
An objective criterion for success of an agent's behavior (often called utility function or reward function).
معيار موضوعي لنجاح سلوك العامل (غالباً يسمى دالة المنفعة أو دالة المكافأة).
[AI_agentsCh2.pdf, p. 9]
PEAS (إطار PEAS)
A framework for specifying the task environment of an agent. Performance measure defines utility and rational actions; Environment includes components and rules; Actuators are what's available to act; Sensors define percepts.
إطار لتحديد بيئة مهمة العامل: مقياس الأداء، البيئة، المحركات، أجهزة الاستشعار.
[AI_agentsCh2.pdf, p. 13]
Fully Observable Environment (بيئة كاملة الرؤية)
The agent's sensors give it access to the complete state of the environment.
أجهزة الاستشعار تعطي العامل وصولاً كاملاً لحالة البيئة.
[AI_agentsCh2.pdf, p. 17]
Partially Observable Environment (بيئة جزئية الرؤية)
The agent cannot see all aspects of the environment.
العامل لا يرى كل جوانب البيئة.
[AI_agentsCh2.pdf, p. 17]
Deterministic Environment (بيئة حتمية)
Changes in the environment are completely determined by the current state of the environment and the agent's action.
تغيرات البيئة تحدد بالكامل بواسطة الحالة الحالية وإجراء العامل.
[AI_agentsCh2.pdf, p. 17]
Stochastic Environment (بيئة عشوائية)
Changes cannot be determined from the current state and the action (there is some randomness).
التغيرات لا يمكن تحديدها من الحالة الحالية والإجراء (يوجد عشوائية).
[AI_agentsCh2.pdf, p. 17]
Strategic Environment (بيئة استراتيجية)
An environment where the outcome depends on the actions of multiple agents, each with their own goals.
بيئة تعتمد نتيجتها على تصرفات عدة عوامل، كل منها له أهدافه الخاصة.
Known Environment (بيئة معروفة)
An environment where the agent knows all the rules and dynamics.
بيئة يعرف فيها العامل جميع القواعد والديناميكيات.
Unknown Environment (بيئة غير معروفة)
An environment where the agent does not know all the rules and must learn them.
بيئة لا يعرف فيها العامل جميع القواعد ويجب عليه تعلمها.
Static Environment (بيئة ثابتة)
An environment that does not change while the agent is deliberating.
بيئة لا تتغير أثناء تفكير العامل.
Dynamic Environment (بيئة ديناميكية)
An environment that can change while the agent is deliberating.
بيئة يمكن أن تتغير أثناء تفكير العامل.
Semi-Dynamic Environment (بيئة شبه ديناميكية)
An environment where the agent's actions affect the environment, but the environment itself does not change otherwise.
بيئة تتأثر فقط بإجراءات العامل ولا تتغير من تلقاء نفسها.
Discrete Environment (بيئة متقطعة)
An environment with a finite number of distinct states and actions.
بيئة تحتوي على عدد محدود من الحالات والإجراءات المميزة.
Continuous Environment (بيئة مستمرة)
An environment with an infinite number of possible states and actions.
بيئة تحتوي على عدد لا نهائي من الحالات والإجراءات الممكنة.
Episodic Environment (بيئة حلقية)
An environment where the agent's experience is divided into atomic episodes, and each episode is independent of the others.
بيئة يتم فيها تقسيم تجربة العامل إلى حلقات مستقلة.
Sequential Environment (بيئة تسلسلية)
An environment where the current decision could affect all future decisions.
بيئة يكون فيها القرار الحالي مؤثراً على جميع القرارات المستقبلية.
Single Agent Environment (بيئة عامل واحد)
An environment with only one agent operating.
بيئة يعمل فيها عامل واحد فقط.
Multi-Agent Environment (بيئة متعددة العوامل)
An environment with multiple agents interacting.
بيئة يتفاعل فيها عدة عوامل معاً.
Simple Reflex Agent (عامل رد الفعل البسيط)
An agent that selects actions based only on the current percept, ignoring the rest of the percept history.
عامل يختار إجراءاته بناءً فقط على الإدراك الحالي ويتجاهل باقي تاريخ الإدراك.
Model-Based Reflex Agent (عامل رد الفعل المعتمد على النموذج)
An agent that maintains some internal state to keep track of aspects of the world it cannot see directly.
عامل يحتفظ بحالة داخلية لمتابعة جوانب العالم التي لا يمكنه رؤيتها مباشرة.
Transition Function (دالة الانتقال)
A function that describes how the environment changes in response to the agent's actions.
دالة تصف كيف تتغير البيئة استجابةً لإجراءات العامل.
State Space (فضاء الحالات)
The set of all possible states in which the agent can be.
مجموعة جميع الحالات الممكنة التي يمكن أن يكون فيها العامل.
Goal-Based Agent (عامل قائم على الهدف)
An agent that acts to achieve specific goals.
عامل يتصرف لتحقيق أهداف محددة.
Utility-Based Agent (عامل قائم على المنفعة)
An agent that acts to maximize a utility function.
عامل يتصرف لتعظيم دالة المنفعة.
Learning Element (عنصر التعلم)
A component of an agent that improves its performance based on experience.
عنصر في العامل يحسن أداءه بناءً على الخبرة.

3 Lecture 3: AI Search

Search Problem (مشكلة البحث)
The problem of designing goal-based agents in known, fully observable, and deterministic environments.
مشكلة تصميم عوامل قائمة على الهدف في بيئات معروفة وكاملة الرؤية وحتمية.
[AI_searchCh3.pdf, p. 2]
Initial State (الحالة الابتدائية)
The starting state description for the search.
وصف الحالة التي يبدأ منها البحث.
[AI_searchCh3.pdf, p. 5]
Actions (الإجراءات)
The set of possible actions available from a given state.
مجموعة الإجراءات الممكنة من حالة معينة.
[AI_searchCh3.pdf, p. 5]
Transition Model (نموذج الانتقال)
A function that defines the new state resulting from performing an action in the current state.
دالة تحدد الحالة الجديدة الناتجة عن تنفيذ إجراء في الحالة الحالية.
[AI_searchCh3.pdf, p. 5]
Goal State (الحالة الهدف)
The state description that the agent aims to reach.
وصف الحالة التي يسعى العامل للوصول إليها.
[AI_searchCh3.pdf, p. 5]
Path Cost (تكلفة المسار)
The sum of step costs to reach a particular state from the initial state.
مجموع تكاليف الخطوات للوصول إلى حالة معينة من الحالة الابتدائية.
[AI_searchCh3.pdf, p. 5]
Optimal Solution (الحل الأمثل)
The sequence of actions (or equivalently a sequence of states) that gives the lowest path cost for reaching the goal.
تسلسل الإجراءات (أو الحالات) الذي يعطي أقل تكلفة مسار للوصول إلى الهدف.
[AI_searchCh3.pdf, p. 5]
State Space (فضاء الحالات)
The set of all possible states of the environment and some states are marked as goal states. It is the number of different states the agent and environment can be in.
مجموعة جميع الحالات الممكنة للبيئة وبعض الحالات تعتبر حالات هدف. هو عدد الحالات المختلفة التي يمكن أن يكون فيها العامل والبيئة.
[AI_searchCh3.pdf, p. 4, 21]
Search Tree (شجرة البحث)
A "what if" tree of possible actions and outcomes (states) superimposed on the state space graph.
شجرة "ماذا لو" للإجراءات والنتائج الممكنة (الحالات) فوق رسم فضاء الحالات.
[AI_searchCh3.pdf, p. 13]
Root Node (العقدة الجذرية)
Represents the initial state in a search tree.
تمثل الحالة الابتدائية في شجرة البحث.
[AI_searchCh3.pdf, p. 13]
Uninformed Search Strategies (استراتيجيات البحث غير المستنيرة)
Search algorithms where the agent is not provided information about how close a state is to the goal state. It blindly searches following a simple strategy until it finds the goal state by chance.
خوارزميات بحث لا تزود العامل بمعلومات عن مدى قرب الحالة من الهدف. يبحث بشكل أعمى باتباع استراتيجية بسيطة حتى يجد الهدف بالصدفة.
[AI_searchCh3.pdf, p. 28]
Breadth-First Search (BFS) (البحث بالعرض أولاً)
An uninformed search strategy that expands the shallowest unexpanded node in the frontier (FIFO).
استراتيجية بحث غير مستنيرة توسع العقدة الأقل عمقاً في الجبهة (FIFO).
[AI_searchCh3.pdf, p. 29]
Uniform-Cost Search (UCS) (البحث بالتكلفة الموحدة)
An uninformed search strategy that expands the node in the frontier with the least path cost from the initial state (Dijkstra's shortest path algorithm).
استراتيجية بحث غير مستنيرة توسع العقدة ذات أقل تكلفة مسار من الحالة الابتدائية (خوارزمية ديكسترا).
[AI_searchCh3.pdf, p. 35]
Depth-First Search (DFS) (البحث بالعمق أولاً)
An uninformed search strategy that expands the deepest unexpanded node in the frontier (LIFO).
استراتيجية بحث غير مستنيرة توسع العقدة الأعمق في الجبهة (LIFO).
[AI_searchCh3.pdf, p. 38]
Iterative Deepening Search (IDS) (البحث بالتعمق التدريجي)
A search strategy that combines the benefits of BFS and DFS by performing a series of depth-limited DFS searches with increasing depth limits.
استراتيجية تجمع بين مزايا البحث بالعرض والبحث بالعمق عبر سلسلة من عمليات البحث المحدودة العمق مع زيادة حدود العمق تدريجياً.
[AI_searchCh3.pdf, p. 44]
Completeness (الإكتمال)
Does it always find a solution if one exists?
هل يجد دائماً حلاً إذا كان موجوداً؟
[AI_searchCh3.pdf, p. 19]
Optimality (المثالية)
Does it always find a least-cost solution?
هل يجد دائماً الحل الأقل تكلفة؟
[AI_searchCh3.pdf, p. 19]
Time Complexity (تعقيد الزمن)
How long does it take?
كم من الوقت يستغرق؟
[AI_searchCh3.pdf, p. 19]
Space Complexity (تعقيد المساحة)
How much memory does it need?
كم من الذاكرة يحتاج؟
[AI_searchCh3.pdf, p. 19]
Branching Factor (عامل التفرع)
Maximum number of successor nodes for a parent in the search tree.
أقصى عدد من العقد الفرعية لعقدة رئيسية في شجرة البحث.
[AI_searchCh3.pdf, p. 19]
Depth (العمق)
Depth of the optimal solution (number of actions needed).
عمق الحل الأمثل (عدد الإجراءات المطلوبة).
[AI_searchCh3.pdf, p. 19]
Maximum Depth (العمق الأقصى)
The number of actions in any path (may be infinite with loops).
عدد الإجراءات في أي مسار (قد يكون لا نهائي في وجود الحلقات).
[AI_searchCh3.pdf, p. 19]
